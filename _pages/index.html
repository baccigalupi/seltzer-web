A lightweight SCSS framework
for modern browsers

Oy vey! In the olden-days, we needed a stiff drink to muster through styling across browsers. In fact, the great designers of thoughtbot olde created buorbon and bitters which have taken us through harder times.

Since then browsers have done a lot to grow up, and we are so happy about it. The box model has evolved so that we can predict the actual size of many tags in our layouts. Many good built-in tools have come to pass. Flexbox has provided us with ways to flow content in the page, and the grid layout allows complicated and dynamic layout design.

Why do we need a modern framework?

CSS remains an incomplete language that continutes to throw upon us the unexpected, with a daunting amount of options and no debugger.

Seltzer is a set of tools and guidelines to help developers navigate the challenges of modern browsers:

  1. Build custom component systems instead of custom CSS
  2. Reduce the complexity, and increase the predictablitiy of modern CSS
  3. Make responsive design easy
